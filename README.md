# 📝 My Notes

A modern and simple note-taking application built with **Flutter**.  
The app allows users to create, edit, delete, and manage notes locally using SQLite with multilingual support.

---

## 📱 Features

- ✨ Beautiful Splash Screen with animations
- 📝 Create new notes
- ✏️ Edit existing notes
- 🗑️ Delete notes with confirmation dialog
- 💾 Local data storage using SQLite
- 🌍 Multi-language support (English & Arabic)
- 🔄 Save language preference using SharedPreferences
- 📱 Clean and responsive user interface
- ⚡ Fast and lightweight performance

---


## 🛠️ Built With

- Flutter
- Dart
- SQLite (sqflite)
- GetX
- SharedPreferences

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Mohammedbanboraik/My-Notes.git
```

Go to the project folder

```bash
cd My-Notes
```

Install dependencies

```bash
flutter pub get
```

Run the application

```bash
flutter run
```
